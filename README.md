Curso Intensivo de Python
Este repositório contém meus exercícios e projetos desenvolvidos durante a leitura do livro "Curso Intensivo de Python: Uma Introdução Prática e Baseada em Projetos à Programação" (3ª Edição, 2023) de Eric Matthes.
📖 Sobre o Livro
O livro é dividido em duas partes principais:
Conceitos Básicos: Fundamentos da linguagem (listas, dicionários, classes, testes).
Projetos: Aplicação prática em desenvolvimento de jogos, visualização de dados e aplicações web.
🗂️ Estrutura do Repositório
Parte I: Fundamentos
capitulo_01/: Configuração do ambiente.
capitulo_02/: Variáveis e tipos de dados simples.
capitulo_03_04/: Listas (trabalhando e manipulando).
capitulo_05/: Instruções if.
capitulo_06/: Dicionários.
capitulo_07/: Entrada de usuário e laços while.
capitulo_08/: Funções.
capitulo_09/: Classes (POO).
capitulo_10/: Arquivos e exceções.
capitulo_11/: Testando o seu código (unittest).
Parte II: Projetos
projeto_1_alien_invasion/: Um jogo de tiro espacial estilo arcade.
projeto_2_data_visualization/: Gráficos e mapas gerados com Matplotlib e Plotly.
projeto_3_learning_log/: Uma aplicação web completa feita com Django.
🚀 Como usar
Certifique-se de ter o Python 3.x instalado.
Clone o repositório:
bash
git clone https://github.com
Use o código com cuidado.

Navegue até a pasta do capítulo ou projeto desejado e execute o script:
bash
python nome_do_arquivo.py
Use o código com cuidado.
